Informe: Encender un foco si la distancia es mayor a 100cm, si es menor prender un foco

1. Introducción

Este informe detalla el diseño y funcionamiento de un circuito electrónico basado en Arduino. El sistema utiliza un sensor ultrasónico para medir la distancia a un objeto. Dependiendo de si la distancia es mayor o menor, el sistema activa una señal LED o una alarma.

2. Componentes Utilizados

  • Placa Arduino Uno
  • Sensor Ultrasónico de Distancia
  • LED
  • Alarma (Buzzer)
  • Resistencia
  • Placa de pruebas (Protoboard)
  • Cables de conexión

3. Funcionamiento y Características Clave


El sistema opera midiendo continuamente la distancia. El sensor ultrasónico emite un pulso de sonido y mide el tiempo que tarda en regresar el eco. Este tiempo se convierte en una distancia en centímetros.

Las conexiones principales de los componentes con la placa Arduino, según la configuración del programa, son:

  • Sensor Ultrasónico (Ping): Conectado al pin digital 12. Este pin se configura dinámicamente como salida para enviar el pulso y como entrada para recibir el eco.
  • LED : Conectado al pin digital 7.
  • Alarma : Conectado al pin digital 6.

La lógica de operación se basa en un umbral de 100 cm:

  • Condición 1: Distancia mayor a 100 cm.
    Si no se detectan objetos dentro del rango de 100 cm, el LED se enciende para indicar un funcionamiento normal, mientras que la alarma permanece apagada.
  • Condición 2: Distancia menor o igual a 100 cm.
    Si un objeto cruza el umbral y se acerca a 100 cm o menos del sensor, el LED se apaga y se activa la alarma, emitiendo un sonido.

El ciclo de medición y comprobación se repite cada 500 milisegundos para proporcionar una respuesta en tiempo real.

5. Conclusión

Este proyecto es un ejemplo práctico de cómo integrar sensores para crear sistemas de respuesta automatizada. Sus aplicaciones pueden variar desde un sistema simple de seguridad, un asistente de estacionamiento para vehículos, hasta un dispositivo para recordar y mantener el distanciamiento en espacios públicos.
