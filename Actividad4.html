Informe: Sistema de inicio de sesión con LCD, LED y alarma

Autor: Josue Jacome

Objetivo

Implementar un sistema de inicio de sesión con Arduino, donde al ingresar la contraseña correcta se encienda un LED y se muestre "Correcto" en la pantalla LCD, y si la contraseña es incorrecta se active una alarma y aparezca "Incorrecto".

Simulación práctica

Marco Teórico

  • Arduino: Microcontrolador que permite controlar periféricos y ejecutar lógicas de programación.
  • LCD: Pantalla usada para mostrar mensajes como "Correcto" o "Incorrecto".
  • LED: Indicador visual para confirmar acceso correcto.
  • Alarma (buzzer): Señal acústica que se activa cuando la contraseña es incorrecta.
  • Teclado matricial: Permite al usuario ingresar la contraseña.

Procedimiento

  1. Configurar la pantalla LCD, teclado matricial, LED y buzzer en Tinkercad.
  2. Programar en Arduino una contraseña predeterminada.
  3. Verificar la entrada del usuario comparándola con la contraseña.
  4. Si la contraseña es correcta, mostrar Correcto y encender el LED.
  5. Si la contraseña es incorrecta, mostrar Incorrecto y activar la alarma.

Conclusiones

El sistema cumple su función de validar accesos mediante contraseña, reforzando el uso combinado de periféricos en Arduino. Esta práctica permite comprender cómo implementar medidas de seguridad básicas en proyectos electrónicos.
